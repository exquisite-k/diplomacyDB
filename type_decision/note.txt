# AI 분석 포털 웹사이트 구현 가이드

## 프로젝트 개요
이 프로젝트는 Figma 디자인을 바탕으로 'AI 분석 포털' 웹사이트를 구현하는 작업입니다.
해당 사이트는 AI 서비스, 트럼프 미국 관세, 인물 관계도, 키워드 연관성, 언급어 등의 다양한 데이터 시각화와 정보를 제공합니다.

## 디렉토리 구조
```
/
├── index.html         # 메인 HTML 파일
├── css/
│   └── style.scss     # SCSS 스타일시트
├── js/
│   ├── main.js        # 메인 자바스크립트 파일
│   ├── chart.js       # amChart5 차트 관련 기능
│   └── slide.js       # Swiper.js 슬라이드 관련 기능
├── images/            # 이미지 파일들
└── note.txt           # 프로젝트 가이드 및 문서
```

## 기술 스택
- HTML5 (하드코딩 방식)
- SCSS (CSS 전처리기)
- JavaScript (ES6+)
- Swiper.js (슬라이드 기능)
- amChart5 (데이터 시각화)

## 구현 가이드
1. **HTML**: 시맨틱 태그를 활용하여 구조적인 HTML 작성
2. **CSS**: SCSS 형식으로 스타일 구분, 일반 CSS 표기법 병행
3. **JavaScript**: 
   - 기능별 함수 분리
   - DOM ready 이벤트에서 초기화
   - Swiper.js로 슬라이드 구현
   - amChart5로 다양한 차트 구현
4. **코드 주석**: 각 주요 섹션과 기능에 설명 주석 추가

## 작업 로그
### 1. 초기 요구사항 및 설정
- Figma 디자인을 기반으로 웹 페이지 구현
- HTML 하드코딩, SCSS 스타일링, 함수형 JavaScript 구현
- Swiper.js 슬라이드, amChart5 차트 통합
- 코드 주석 및 가이드 문서화 

### 2. HTML 구조 작업
- 시맨틱 태그를 활용한 웹 페이지 구조 작성
- 헤더, 네비게이션, 메인 배너, 서비스 소개 등 섹션 구성
- Swiper.js 및 amChart5 라이브러리 통합
- BEM 방법론을 활용한 CSS 클래스명 구성

### 3. SCSS 스타일링
- 변수 설정 (색상, 여백, 폰트 등)
- 컴포넌트별 스타일 분리 및 구조화
- 반응형 레이아웃 구현 (미디어 쿼리 활용)
- 애니메이션 및 전환 효과 적용

### 4. 자바스크립트 기능 구현
- Swiper.js를 활용한 메인 배너 및 뉴스 슬라이드 구현
- amChart5를 활용한 인물 관계도, 키워드 언급량, 언급어 차트 구현
- 데이터맵 토글 기능, 검색 기능 등 UI 인터랙션 구현
- 데이터 수집 현황 차트 시각화
